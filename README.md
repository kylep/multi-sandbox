# multi-sandbox

Experiments, half-baked ideas, and archived projects. **Nothing here is
maintained.** No quality bar, no branch protection, no dependency
hygiene, no promises — commit straight to `main`.

Actively maintained work lives in [kylep/multi](https://github.com/kylep/multi),
which is treated as production: PRs to main, accurate docs, zero-vuln
policy. When a project there stops being maintained it migrates here;
if something here graduates, it moves to `multi` and gets brought up to
that standard.

Most projects arrived in the 2026-07-04 demotion from `multi`
(kylep/multi PR: "demote unmaintained projects to multi-sandbox").
Their full git history remains in `multi`'s history.
